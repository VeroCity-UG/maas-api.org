Welcome to MaaS-API project

The purpose of the MaaS-API project is to provide an open set of APIs for driving Mobility-as-a-Service ecosystem forward in a community fashion. The project documents the APIs and provides a reference implementation for anyone to use and integrate to.

MaaS Business Process Engine with the system integration APIs

Interface Documentation

MaaS Transport Service Provider (TSP) APIs are an open source project under the MIT license - anyone can use it as part of their open or closed source solution. We do this in order to kick-start the MaaS ecosystem with the functional API to implement and build upon. The API is being used as key part of the MaaS Finland solution so by implementing it, TSP's are well on their way to being part of the rapidly growing MaaS movement, regardless of who operates the end-user service in the area.

Want to be included in this list? MaaS Finland is open for having integrators working with TSPs around the world get certified as MaaS Cetrified Integrator Partners. It's quite simple, really:

  • Submit a testable TSP adapter to us or
  • Get your pull request approved or
  • Give our maintainers @pippuri or @laurisvan a shout and let them know how you could help improve our offering
